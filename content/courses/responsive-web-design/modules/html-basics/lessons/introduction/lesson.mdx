# Introduction to Semantic HTML

HTML is more than a collection of boxes. Semantic elements communicate meaning to browsers, assistive technology, search engines, and other developers.

In this lesson, you will build a small landing page using `header`, `main`, `section`, and `footer` elements.

<Callout type="tip">
  Semantic HTML is the foundation for accessibility and maintainable frontends.
</Callout>

## Challenge

Create a simple profile card page with semantic layout landmarks and one call-to-action link.
